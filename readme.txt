A simple & lightweight, multilingual WordPress plugin to collect your users 5 star rating like feedback.

== Description ==

Are you interested in your user's Feedback? Would you like to know, if they like your product or not? Create an easy and straightforward Quiz like Feedback form, and place it anywhere on your website: on a page, on the sidebar, in a lightbox, or wherever you prefer by the shortcode:
[upd8-feedback id="FORM-ID"]

== Installation ==

How to install this plugin?

1. Upload the plugin files to the `/wp-content/plugins/upd8-feedback` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Feedback menu to set up your first Form
4. Place shortcode: [upd8-feedback id="FORM-ID"] anywhere on your website


== Frequently Asked Questions ==

= Where can I check the stats of my form fills? =

All the form fills and their averages are displayed on the bottom of form page.

= How to prevent duplicated form fills from the same user? =

There are two ways to do this, both of them can be turned on on the Form edit page.
"Prevent duplicated form fills (Cookie)" uses a Cookie variable to check if the user has filled in the form already.
"Prevent duplicated form fills (IP)" allows only one Form fill per IP address.

= How to customise colors of your form? =
You can simply customise colors of your stars on the Form edit page.
Buttons are HTML button tags, so those should suit automatically your theme, by it's CSS.

= How to enable Skip question button? =
If you want to let your users to skip a question without answering it, check the checkbox "Display skip button" on your Form edit page. These answers won't be counted in the stats.

= How to enable "Slide to next automatically"? =
This can be easily set on your Form edit page. If it's enabled, the plugin will show the next question right after the user answered one, without have to press the next button.
